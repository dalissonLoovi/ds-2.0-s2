## AI-READY

AI-READY COMPONENT: List is a vertical container that composes one or more ListItem rows through an items SLOT. Use type=plain for an embedded page list surface and type=dropdown for an elevated menu/panel surface. Defaults in the component set: type=dropdown, showScroll=false. Props: items is the SLOT for ListItem instances (DividerHorizontal may separate groups); showScroll toggles the local right-edge scroll affordance when content overflows. Do not put ListActionDropdownItem inside List — use ListItem only. Do not use List as a select field, combobox, navigation drawer, table, or action menu panel; for action menus use ListActionDropdown. type=dropdown is the panel surface only — the external trigger owns aria-expanded / open state. Token rule: use existing DS kebab-case variables only — color/background-surface/0, border/radius/0 (plain) or border/radius/200 (dropdown), elevation/2 on dropdown, color/palette/graph + border/radius/full for scroll, plus nested ListItem and DividerHorizontal tokens. Accessibility: map plain lists to role=list with ListItem as listitem; map dropdown panels to role=menu or listbox depending on selection pattern; keyboard and focus belong to the product implementation; scroll is visual-only. React mapping: List(\{ type, showScroll, children \}) where children map to the items slot.

## Rules

- **slotListItemOnly:** Compose items SLOT with ListItem only — not ListActionDropdownItem
- **plainVsDropdown:** type=plain = page list surface; type=dropdown = elevated panel surface
- **triggerExternal:** type=dropdown has no internal trigger — parent owns open/aria-expanded
- **vsActionMenu:** For action menus use ListActionDropdown
- **vsSelect:** Do not use as select field, combobox, navigation drawer or table
- **scroll:** Absolute right-edge thumb; visible via showScroll when list overflows
- **defaults:** type=dropdown, showScroll=false
