## AI-READY

AI-READY COMPONENT: ListActionDropdown is a menu surface container for grouped list actions opened by an external trigger. Use it as the menu panel only — compose action rows with ListActionDropdownItem and separate groups with DividerHorizontal inside the slot. Props: showScroll toggles the local scroll affordance (right-edge thumb) when the list overflows; showDivider toggles the divider instance inside the default slot composition; slot replaces the default menu content with a custom ListActionDropdownItem / DividerHorizontal composition. Do not use remote menu kits, tooltips, keyboard shortcut chrome, icons or flags. Do not use ListActionDropdown as a page list, select field, tooltip, modal, navigation drawer or standalone button. Token rule: use existing DS kebab-case variables only — color/background-surface/0, color/elevation/5, color/palette/graph, spacing/0|050|100|200|300|800, border/radius/200|full, border/width/025, plus nested ListActionDropdownItem and DividerHorizontal tokens. Accessibility: map the container to role=menu when rendered as a menu; the external trigger controls aria-expanded and aria-controls; items map to role=menuitem (or option semantics if selectable); divider is role=separator or aria-hidden when decorative; keyboard behavior belongs to the menu implementation (ArrowUp/ArrowDown, Enter/Space, Escape). React mapping: ListActionDropdown(showScroll, showDivider, children) where children map to slot.
